# FNAF-1-Python-autoplay-bot

This is a beginner project which is inspired from already existing autobots.
The script could do some inaccurate reading but it can beat 4/20 with enough tries

# Installation
1. Download the files in a folder
2. Install Python 3.12.10
3. Run the command `pip install -r requirements.txt` in the visual studio code terminal
4. Open FNAF 1 which has the title "Five Nights at Freddy's" and make sure no other window open has the same title
5. Wait for the bot to play, press "x" in case you want to stop the bot
